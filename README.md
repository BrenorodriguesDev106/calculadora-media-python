# Calculadora de Média em Python

Projeto desenvolvido para praticar lógica de programação utilizando Python.

## 📌 Funcionalidades
- Recebe notas digitadas pelo usuário
- Calcula a média
- Informa se o aluno foi aprovado ou reprovado

## 🛠 Tecnologias utilizadas
- Python 3

## 🎯 Objetivo
Projeto desenvolvido para praticar estruturas condicionais, entrada de dados e lógica de decisão em Python.

## ▶️ Como executar
1. Baixe o arquivo `calculadora_media.py`
2. Execute no Python
3. Digite as notas quando solicitado
4. 📎 Status
Projeto finalizado como parte dos meus estudos iniciais em Python.
